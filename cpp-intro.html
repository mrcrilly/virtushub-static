Want to be a video game programmer, but not too sure where to start? Throughout this series you’ll learn the fundamentals and principles of C++ programming and how you can use it within Unreal Engine 4. You’ll learn the language just like any other, breaking down the principles of object oriented programming, event driven programming and correct syntax. You’ll learn everything you need to know such as working with variables, functions, loops & more!
